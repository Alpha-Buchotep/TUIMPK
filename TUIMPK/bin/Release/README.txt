 TUIMPK = TCP / UDP / ICMP Multi Port Knocker
----------------------------------------------
 Version: 1.0.6.0
 Date: 2023-10-17
----------------------------------------------

-------------
 Description
-------------

With this app You can knock port(s) on supported devices like Mikrotik router / switch, IPTables etc.

--------------
 Features:
--------------

- Free
- Portable
- You can choose protocol: UDP/TCP/ICMP
- You can choose port for UDP/TCP
- You can set "password" for UDP (Layer7)
- You can save / load configs (encrypted with internal or Your password AES256)
- You can adjust delay between packets (0.5 - 5 sec)
- You can save config file password (stored in app settings, AES256 enrcrypted)
- Every field content is hidden for maximum security (double click to show)
- Two version available (music / no music)
- Eight background (click to program icon (on top left)

--------------
 Requirements
-------------- 

- Windows OS (Vista / W7 / W8/ W10/ W11)
- Microsoft .NET Framework 4.5
